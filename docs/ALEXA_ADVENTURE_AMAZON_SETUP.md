# إعداد Amazon — مغامرة النجوم (إجابات 1، 2، 3)

## نظام الإجابة

الطفل **لا يقول** كلب أو موزة. يقول **رقم أو ترتيب** الخيار:

| الإجابة الصوتية | المعنى |
|-----------------|--------|
| `واحد` / `1` / `الأول` | الجواب الأول |
| `اثنان` / `2` / `الثاني` | الجواب الثاني |
| `ثلاثة` / `3` / `الثالث` | الجواب الثالث |

**مثال (سؤال الكلب):**  
Alexa: «الجواب الأول: قطة. الثاني: كلب. الثالث: تفاحة.»  
الطفل: **«اثنان»** أو **«الثاني»**

**الجاهزية:** «نعم» أو «جاهز» (+ `AMAZON.YesIntent`).

---

## AnswerIntent — slot `answer`

| الإعداد | القيمة |
|---------|--------|
| النوع | **`AMAZON.NUMBER`** (مفضل) |

أو **Custom slot** بالقيم:

```
واحد اثنان ثلاثة 1 2 3 الأول الثاني الثالث اول ثاني ثالث
```

### Sample utterances

```
{answer}
الجواب {answer}
الجواب رقم {answer}
رقم {answer}
واحد
اثنان
ثلاثة
الأول
الثاني
الثالث
1
2
3
```

ثم **Save** → **Build Model**.

## التحقق

بعد Build، عند سؤال الخيارات قل **«اثنان»** — JSON Input يجب أن يظهر:

```json
"intent": {
  "name": "AnswerIntent",
  "slots": {
    "answer": { "value": "2" }
  }
}
```

**ليس** `AMAZON.FallbackIntent` بدون slots.
